## Technology Stack

### Core Libraries
- **Node.js** - JavaScript runtime (v18+)
- **TypeScript** - Type-safe JavaScript
- **better-sqlite3** - Local database
- **Crypto (Node.js built-in)** - Cryptography
- **libp2p** - P2P networking library

### Cryptography Implementations
- **AES-256-GCM** - Symmetric encryption
- **ECDH (P-256)** - Key exchange
- **ECDSA-SHA256** - Digital signatures
- **HKDF-SHA256** - Key derivation
- **PBKDF2-SHA256** - Password hashing

### Architecture Inspiration
- **Session Messenger** (https://getsession.org/) - Privacy-focused architecture
- **Loki Network** - P2P infrastructure concepts
- **Signal Protocol** - Encryption standards and best practices

---## Design Principles

This project was built with the following principles:

1. **Privacy First** - End-to-end encryption by default
2. **Decentralized** - No central point of failure
3. **Security** - Industry-standard cryptography
4. **Clean Code** - TypeScript with full type safety
5. **Well Documented** - Comprehensive guides and API docs
6. **Educational** - Learn P2P and E2E encryption concepts
